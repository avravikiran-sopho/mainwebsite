General Rules

  • The competition must be played individually.
  • The problem statement is divided into 3 parts - easy , moderate and difficult.
  • All the question in easy part carries 5 marks,moderate 10 marks and difficult 30 marks
  • Plagiarism will result in disqualification.
  • All the relevant information regarding The question is given in the question itself.
  • The organisers verdict is final.

Submission Rules

  • The Matlab code and the answer should be submitted as a zip file in the portal itself.
  • Participants are expected to follow the naming convention.
  • Matlab code and the answer of each question should be given name according to the question number itself.
    example: answers code for question 1 should be submitted as "< question_1_code>" and answer as "< question_1_ans />"
  • submission is not allowed after the set deadline.
  • The submission is one time submission, check properly before submitting.
  • The organiser verdict is final
