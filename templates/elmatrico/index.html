EL MATRICO

General Rules

  • The competition must be played individually.
  • The problem statement is divided into 3 parts - easy , moderate and difficult.
  • All the question in easy part carries 5 marks,moderate 10 marks and difficult 30 marks
  • Plagiarism will result in disqualification.
  • All the relevant information regarding The question is given in the question itself.
  • The organisers verdict is final.
  • Cash prizes will be given to top two contestants on leaderboard.

Submission Rules

  • The Matlab code(.m files) and the answers(output files like Graphs, Images and wav files etc) of all the questions should be submitted as a zip file in the portal itself.
  • Participants are expected to follow the naming convention.
  • Matlab code and the answer of each question should be given name according to the question number itself . Example : answers for question 1 should be submitted as < question_1_code.m > and answer as < question_1_ans >
  • Zip file should be named as < ELANID.zip >
    • Example: EN18IITH00001.zip
  • Submission is not allowed after the set deadline.
  • It is one time submission, check properly before submitting.
    • The organiser verdict is final.

Event starts on 29th Dec 2017.
